To generate a Vivado project:
	1) Start Vivado
	2) Execute the script "create_vivado_project.tcl" 
		Ex.: source <user folder>/CoaxlinkQuadCxp12_1cam/03_scripts/create_vivado_project.tcl
 
	Obs.: The script "create_vivado_project.tcl" creates a new Vivado
	      project at the folder: <user folder>/CoaxlinkQuadCxp12_1cam/07_vivado_project
